¿Cómo configurar el Apps Script de escritura?
  1. En Google Sheets → Extensiones → Apps Script.
  2. Creá una función doGet(e) que lea e.parameter.action y opere sobre la hoja.
  3. Implementar → Nueva implementación → Aplicación web. Ejecutar como: yo. Acceso: Cualquier usuario.
  4. Copiá la URL y pegála en GOOGLE_SCRIPT_URL dentro de app.js.
  5. El script recibe: action (save / delete), id, nombre, precio, descripcion, imagenes, estado.
