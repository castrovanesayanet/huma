Configuración pendiente

Para que el formulario guarde datos en Google Sheets, reemplazá GOOGLE_SCRIPT_URL en app.js con la URL de tu Web App de Apps Script.

Ver instrucciones rápidas
  1. En la planilla de Google Sheets → Extensiones → Apps Script.
  2. Pegá el código del Web App (doPost) que busca el ID y actualiza la fila, o la agrega si no existe.
  3. Hacé clic en Implementar → Nueva implementación → Aplicación web.
  4. Ejecutar como: yo. Acceso: Cualquier usuario.
  5. Copiá la URL generada y reemplazá el placeholder en app.js.
